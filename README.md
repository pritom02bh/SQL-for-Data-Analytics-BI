## SQL-for-Data-Analysis

MySQL Files &amp; Documentations                                                                   
***By Pritom Bhowmik***


### Data Definition Language (DDL)          
       - CREATE
       - ALTER
       - DROP
       - RENAME
       - TRUNCATE
            
### Data Manipulation Language (DML)
       - SELECT ... FROM
       - INSERT INTO ... VALUES
       - UPDATE ... SET ... WHERE
       - DELETE ... FROM ... WHERE
       
       
### Data Control Language (DCL)
       - GRANT
       - REVOKE
       
### Transaction Control Language (TCL)
       - COMMIT
       - ROLLBACK
