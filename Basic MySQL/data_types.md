### String Data Type

                        
                    String Data Type             |   Storage    |     Example     |    Maximum size (bytes)   |       
               ----------------------------------|--------------|-----------------|---------------------------|--------------------------------------------------------    
                    character (CHAR)             |   fixed      |    CHAR(5)      |     255                   |   50% faster
               ----------------------------------|--------------|-----------------|---------------------------|--------------------------------------------------------     
                    variable character(VARCHAR)  |  variable    |   VARCHAR(5)    |    65,535                 |   a lot more responsive to the data value inserted
                                                 |              |                 |                           |
